# MovieApp
Movie app developed by VIPER design pattern and 3rd party libraries. 

Technologies:
- UITableView
- UIScrollView
- UIStackView
- No storyboards - programmatic UI
- Kingfisher
- UIStackView
- TMDB API
- URL Session
- VIPER design pattern
- UIKit
- Delegation pattern
